# 16-1. 점근 분석 심화 — 실습

> **참고** 이 실습은 제출물이 아니라, 점근 표기의 정의와 분할상환 비용을 직접 계산하고 측정해보는 과정입니다.

## 실습 1 — O·Ω·Θ를 손으로 판별하기

- [ ] $f(n) = 3n^2 + 5n + 10$에 대해 $f(n) = O(n^2)$을 만족하는 상수 $c$와 $n_0$의 구체적인 값 한 쌍을 직접 찾아 부등식이 성립하는지 종이에 검산해보세요.
- [ ] 같은 $f(n)$에 대해 $f(n) = \Omega(n^2)$을 만족하는 $c, n_0$도 찾아, $f(n) = \Theta(n^2)$임을 두 부등식으로 보이세요.
- [ ] $f(n) = O(n^3)$은 참이지만 $f(n) = \Theta(n^3)$은 거짓임을 정의로 설명하는 문장을 세 줄 이내로 적어보세요.

## 실습 2 — 동적 배열의 분할상환 비용 실측하기

- [ ] 배열 용량을 꽉 차면 **2배로** 늘리는 동적 배열을 직접 구현하고, `append`마다 발생한 복사 횟수를 누적 변수에 기록하세요.
- [ ] 원소를 100,000개 넣은 뒤 (총 복사 횟수 ÷ 100,000)을 계산하고, 그 값이 2 미만인지 확인하세요.
- [ ] 성장 배율을 2배 대신 **매번 +1**로 바꿔 같은 실험을 반복하고, 1회당 복사 횟수가 어떻게 달라지는지 두 결과를 표로 비교하세요.
- [ ] 성장 배율을 1.5배로 바꿔 세 번째 실험을 하고, 총 복사 횟수와 최대 메모리 낭비량이 2배 방식과 어떻게 다른지 수치로 적어보세요.

## 실습 3 — 마스터 정리 적용 연습

- [ ] $T(n) = 2T(n/2) + n$, $T(n) = 4T(n/2) + n$, $T(n) = T(n/2) + 1$, $T(n) = 3T(n/4) + n^2$ 네 식에 각각 마스터 정리를 적용해 $\log_b a$를 계산하고 어느 경우(1/2/3)에 해당하는지 표로 정리하세요.
- [ ] 네 식 중 하나를 골라 재귀 트리를 3단계까지 직접 그리고, 층별 비용의 합이 마스터 정리 결과와 일치하는지 확인하세요.
- [ ] $T(n) = T(n/3) + T(2n/3) + n$에 마스터 정리를 적용할 수 없는 이유를 한 문장으로 적어보세요.

## 실습 4 — 시간-공간 트레이드오프 측정하기

- [ ] 피보나치 수를 (a) 순수 재귀, (b) 메모이제이션, (c) 반복 세 가지로 구현하세요.
- [ ] $n = 30, 35, 40$에 대해 세 구현의 실행 시간을 측정해 표로 기록하고, (a)만 급격히 느려지는 지점을 확인하세요.
- [ ] (b)와 (c)의 최대 메모리 사용량을 비교하고, 시간복잡도가 같은데도 공간복잡도가 다른 이유를 설명해보세요.

> **팁** 실습 2가 이 강의의 핵심입니다. "가끔 비싼 연산이 있어도 전체 평균은 상수"라는 감각은 수식으로 읽을 때보다 복사 횟수를 직접 세어볼 때 훨씬 확실하게 남습니다. 성장 배율을 바꿔가며 총 비용이 선형에서 제곱으로 무너지는 순간을 꼭 직접 목격하세요.
