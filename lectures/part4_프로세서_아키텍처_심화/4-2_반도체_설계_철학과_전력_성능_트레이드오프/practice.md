# 4-2. 실습 — 전력 공식과 실행 방식 트레이드오프

> **참고** 이 실습은 제출물이 아니라, 전력 공식을 자신의 기기에서 체감해보는 과정입니다.

## 1. 전력 공식으로 발열 설명하기

- [ ] 본인 노트북이나 스마트폰의 성능/배터리 모니터링 앱(또는 작업 관리자)을 열어, 가벼운 작업(문서 작성)과 무거운 작업(동영상 인코딩, 3D 게임) 중일 때 CPU 사용률과 (가능하다면) 클럭 속도, 온도를 각각 기록해보세요.
- [ ] 두 상황의 차이를 P = C·V²·f 공식에서 어떤 변수가 바뀐 결과로 볼 수 있을지 설명해보세요.

## 2. 전압과 클럭의 지렛대 비교하기

- [ ] 전압을 10% 낮췄을 때와 클럭을 10% 낮췄을 때, 전력이 각각 몇 퍼센트 줄어드는지 공식을 이용해 직접 계산해보세요.
- [ ] 왜 DVFS가 클럭만 낮추지 않고 전압까지 함께 낮추는지, 계산 결과를 근거로 설명해보세요.

## 3. 설계 선택 시뮬레이션

- [ ] 만약 본인이 배터리 4시간짜리 저가형 노트북을 설계해야 한다면, 비순차 실행 코어와 순차 실행 코어 중 어느 쪽을 주력으로 선택할지, 그리고 그 이유를 이번 강의의 개념을 사용해 한 문단으로 정리해보세요.
- [ ] 반대로 배터리 걱정이 없는 데스크톱용 CPU를 설계한다면 어떤 선택이 합리적일지도 함께 적어보세요.

## 4. 스스로 점검하기

아래 질문에 막힘없이 답할 수 있는지 확인해보세요.

- [ ] P = C·V²·f 공식에서 C, V, f가 각각 무엇을 의미하는지 물리적으로 설명할 수 있다.
- [ ] 전압이 왜 클럭보다 강력한 전력 절감 지렛대인지, 제곱이라는 단어를 사용해 설명할 수 있다.
- [ ] 비순차 실행과 순차 실행의 차이를 재정렬 버퍼라는 단어를 사용해 설명할 수 있다.
- [ ] 빅리틀 구조가 왜 평균 전력을 낮추는지 한 문단으로 설명할 수 있다.
