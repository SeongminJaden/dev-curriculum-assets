# 10-1 실습 — CI/CD 파이프라인의 원리

> **참고** 제출물이 아니라, 오늘 배운 CI/CD 개념을 실제 파이프라인 설계에 적용해보는 실습입니다.

## 실습 1. 나만의 파이프라인 설계하기

- [ ] 최근에 만들어본(또는 만들고 싶은) 개인 프로젝트를 하나 정해주세요.
- [ ] 그 프로젝트를 배포한다고 가정하고, 빌드 → 테스트 → 배포 단계에서 각각 어떤 작업이 자동으로 실행되어야 할지 적어보세요.
- [ ] 각 단계가 실패했을 때 파이프라인이 어떻게 동작해야 할지(다음 단계로 넘어가야 할지, 멈춰야 할지) 정해보세요.

## 실습 2. "제 컴퓨터에서는 잘 되는데요" 원인 찾기

- [ ] 개발 환경과 운영 환경이 다를 수 있는 항목을 최소 4가지 적어보세요. (예: 언어 버전, 환경 변수 등)
- [ ] 그중 하나를 골라, CI 파이프라인의 어느 단계에서 그 차이를 미리 발견할 수 있을지 설명해보세요.

## 실습 3. 지속적 전달 vs 지속적 배포 선택하기

- [ ] 아래 두 서비스 중 하나를 골라, 지속적 전달과 지속적 배포 중 어느 방식이 더 적합할지 근거와 함께 정해보세요.
  - [ ] 사내 직원만 쓰는 간단한 재고 조회 대시보드
  - [ ] 수백만 명이 사용하는 결제 서비스의 결제 승인 로직
