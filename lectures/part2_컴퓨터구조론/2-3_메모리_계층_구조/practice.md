# 2-3. 실습 — 메모리 계층 구조

> **참고** 이 실습은 제출물이 아니라, 적중률과 AMAT 공식을 직접 손으로 계산해보며 감을 잡는 과정입니다.

## 1. AMAT 직접 계산하기

- [ ] L1 캐시의 적중시간이 1사이클, 적중률이 97%(미스율 3%), 미스 페널티(L2까지 가는 시간)가 12사이클이라고 할 때 AMAT을 계산해보세요.
- [ ] 같은 조건에서 적중률이 90%로 떨어지면 AMAT이 얼마나 늘어나는지 계산하고, 몇 퍼센트 나빠졌는지 구해보세요.
- [ ] 위 두 결과를 비교해, "적중률이 조금만 떨어져도 체감 성능이 크게 나빠진다"는 본문의 설명이 실제로 맞는지 확인해보세요.

## 2. 지역성 원리로 코드 설명하기

- [ ] 2중 for문으로 2차원 배열을 순회하는 코드를 하나 떠올려 보세요(또는 직접 짧게 작성해보세요).
- [ ] 배열을 행(row) 우선으로 순회하는 것과 열(column) 우선으로 순회하는 것 중 어느 쪽이 공간적 지역성을 더 잘 활용할지 추론해보세요.
- [ ] 그렇게 추론한 이유를 "캐시 라인 단위로 데이터를 가져온다"는 개념을 사용해 설명해보세요.

## 3. 계층 구조를 나만의 비유로 만들기

- [ ] 본문의 "책상-책장-도서관" 비유 대신, 여러분만의 일상 비유를 하나 만들어보세요.
- [ ] 그 비유에서 레지스터, 캐시, 주기억장치, 보조기억장치에 각각 대응하는 요소를 표로 정리해보세요.

## 4. 스스로 점검하기

- [ ] 메모리 계층을 CPU에서 가까운 순서대로 빠짐없이 말할 수 있다.
- [ ] 시간적 지역성과 공간적 지역성의 차이를 예시와 함께 설명할 수 있다.
- [ ] AMAT 공식의 세 항(적중시간, 미스율, 미스 페널티)이 각각 무엇을 뜻하는지 말할 수 있다.
