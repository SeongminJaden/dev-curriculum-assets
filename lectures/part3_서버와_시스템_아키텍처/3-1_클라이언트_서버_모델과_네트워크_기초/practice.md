# 3-1. 클라이언트-서버 모델과 네트워크 기초 — 실습

> **참고** 실습은 제출물이 아니라, 오늘 배운 계층 구조와 핸드셰이크를 직접 관찰해보는 과정입니다.

- [ ] 브라우저의 개발자 도구(Network 탭)를 열고 아무 웹사이트나 새로고침 해보세요.
- [ ] 요청 하나를 선택해 "Timing" 정보를 확인하고, 연결 수립(Connecting/SSL)에 걸린 시간과 실제 응답(Waiting/Content Download)에 걸린 시간을 비교해 적어보세요.
- [ ] 자신이 자주 쓰는 서비스 하나를 골라, 어떤 기능이 TCP(신뢰성 필요)를 쓸 것 같고 어떤 기능이 UDP(속도 우선)를 쓸 것 같은지 이유와 함께 정리해보세요.
