# 3-3. 가상화와 컨테이너 — 실습

> **참고** 실습은 제출물이 아니라, 오늘 배운 가상화 원리를 손으로 직접 확인해보는 과정입니다.

- [ ] 자신의 컴퓨터 또는 클라우드 프리티어 계정에 Docker를 설치하고, `docker run hello-world`를 실행해 컨테이너가 몇 초 만에 시작되는지 체감해보세요.
- [ ] `docker ps`와 `docker images` 명령으로 실행 중인 컨테이너와 로컬에 저장된 이미지 목록을 확인하고, 이미지 크기가 몇 MB인지 기록해보세요.
- [ ] 같은 이미지로 컨테이너를 2개 이상 동시에 실행해보고, 두 컨테이너가 서로의 파일시스템을 볼 수 없다는 사실(네임스페이스 격리)을 명령어로 확인해보세요.
- [ ] VM(가상 머신)을 하나 부팅해본 경험이 있다면, 그때 걸린 부팅 시간과 방금 실행한 컨테이너의 시작 시간을 비교해 표로 정리해보세요.
