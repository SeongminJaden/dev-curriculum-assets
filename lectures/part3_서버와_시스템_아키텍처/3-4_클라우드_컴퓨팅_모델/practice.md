# 3-4. 클라우드 컴퓨팅 모델 — 실습

> **참고** 실습은 제출물이 아니라, 오늘 배운 책임 분리 모델을 실제 서비스에 대입해보는 과정입니다.

- [ ] 평소 사용하는 클라우드 서비스나 도구를 3개 이상 나열하고, 각각이 IaaS·PaaS·SaaS 중 어디에 해당하는지 분류해보세요.
- [ ] 분류한 서비스 중 하나를 골라, 자신(또는 회사)이 직접 관리해야 하는 부분과 제공자가 관리해주는 부분을 표로 나눠 정리해보세요.
- [ ] 같은 애플리케이션을 IaaS로 운영할 때와 PaaS로 운영할 때, 서버 장애 발생 시 누가 무엇을 먼저 확인해야 하는지 비교해보세요.
- [ ] 트래픽이 급증하는 가상의 서비스를 하나 상상하고, 온프레미스로 대응할 때와 클라우드 오토스케일링으로 대응할 때 걸리는 시간과 비용 차이를 논리적으로 추정해 적어보세요.
