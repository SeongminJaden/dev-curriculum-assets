# 19-4. 비트 연산과 2의 보수 — 실습

> **참고** 이 실습은 제출물이 아니라, 비트 연산자의 동작과 정수 표현의 경계를 직접 눈으로 확인하는 과정입니다.

## 실습 1 — 비트 연산자 동작 확인하기

- [ ] `a = 0b1100`, `b = 0b1010`에 대해 `a & b`, `a | b`, `a ^ b`, `~a`를 계산하고, 결과를 `format(x, '08b')`로 2진수 문자열로 출력해보세요.
- [ ] `1 << i`를 `i = 0`부터 `7`까지 출력해 각각이 어떤 10진수인지 표로 정리하고, 2의 거듭제곱과 일치하는지 확인해보세요.
- [ ] Python에서 `~5`가 왜 `-6`인지 2의 보수 정의($\lnot x = -x - 1$)로 설명하고, 다른 값 3개로도 성립하는지 확인해보세요.

## 실습 2 — 산술 시프트와 논리 시프트 구분하기

- [ ] JavaScript(또는 Java)에서 `-8 >> 1`과 `-8 >>> 1`을 각각 출력하고 결과가 왜 다른지 비트 패턴으로 설명해보세요.
- [ ] 같은 언어에서 `-1 >>> 0`의 값을 출력하고, 그 값이 왜 4294967295인지 32비트 부호 없는 정수 범위와 연결해 적어보세요.
- [ ] 양수 `8`에 대해 두 시프트가 같은 결과를 내는지 확인하고, 차이가 음수에서만 나타나는 이유를 한 문장으로 정리해보세요.

## 실습 3 — XOR의 성질 활용하기

- [ ] 리스트에서 하나만 홀수 번 등장하는 값을 XOR 누적으로 찾는 함수를 작성하고, `[4,1,2,1,2]`로 검증해보세요.
- [ ] `a ^ a == 0`과 `a ^ 0 == a`를 임의의 정수 5개로 `assert` 검증하고, `(a^b)^c == a^(b^c)`도 확인해보세요.
- [ ] 정수 리스트 3개를 XOR해 "패리티" 값을 만든 뒤, 원본 중 하나를 지우고 나머지와 패리티를 XOR해 지운 값이 복원되는지 확인해보세요.

## 실습 4 — 비트마스크와 오버플로

- [ ] 원소 4개짜리 리스트에 대해 `for mask in range(1 << 4)`로 모든 부분집합을 출력하고, 총 16개인지 세어보세요.
- [ ] `mask &= mask - 1` 반복으로 켜진 비트 수를 세는 함수를 작성하고, `bin(mask).count('1')`과 여러 값에서 결과가 일치하는지 확인해보세요.
- [ ] Java나 C에서 `int max = 2147483647; max + 1`을 출력해 최솟값으로 순환하는 것을 확인하고, `(low + high) / 2`와 `low + (high - low) / 2`를 `low = 2000000000`, `high = 2100000000`으로 각각 계산해 결과 차이를 기록해보세요.

> **팁** 실습 4의 마지막 항목은 Python으로는 재현되지 않습니다(정수가 임의 정밀도이기 때문). 32비트 정수를 흉내 내려면 `((low + high) // 2) & 0xFFFFFFFF` 후 부호 해석을 직접 하거나, Java·C·Rust 같은 고정 폭 정수 언어에서 확인하세요.
